> **RozgarConnect is a breakthrough platform to digitally transform how India finds, hires and mobilizes local talent.**

## Why Rozgar_Connect Exists
The current job platforms often ignore the people who do blue-collar jobs, such as carpentry and gardening. These individuals may not have the opportunity to apply for high-tech jobs on platforms like LinkedIn, and as a result, they miss out on potential employment opportunities. Rozgar_Connect aims to bridge this gap by providing a platform specifically designed for these individuals, enabling them to find and apply for local jobs that match their skills and preferences. By doing so, Rozgar_Connect seeks to empower this often-overlooked segment of the workforce and provide them with a means to earn a living wage.

The problem of blue-collar job seekers being left behind by traditional job platforms is a significant one. Many of these individuals lack the skills or experience required to compete in the high-tech job market, and as a result, they are often relegated to lower-paying, less stable employment. Rozgar_Connect seeks to change this by providing a platform that is tailored to the needs of blue-collar workers, allowing them to connect with local employers and find employment opportunities that match their skills and experience.

By providing a platform for blue-collar workers to find employment, Rozgar_Connect has the potential to make a significant impact on the lives of these individuals. Not only can it provide them with a means to earn a stable income, but it can also help to promote economic growth and development in local communities. By connecting blue-collar workers with local employers, Rozgar_Connect can help to stimulate economic activity and promote job creation, which can have a positive impact on the overall economy.

## ✨ Features
**Hyperlocal job matching** — Rozgar_Connect's hyperlocal job matching feature allows users to connect with vendors or customers within a 15 km radius, providing access to local job opportunities. This feature is designed to help blue-collar workers find employment opportunities that are close to their location, reducing the need for lengthy commutes and making it easier for them to balance work and family responsibilities.

**Chat** — The chat feature in Rozgar_Connect enables job finders and vendors to connect with each other, facilitating communication and negotiation. This feature allows users to discuss job requirements, salaries, and other details, making it easier for them to find employment opportunities that match their skills and preferences.

## 🚀 Quick Start
To get started with Rozgar_Connect, you will need to have the following prerequisites installed:
* Node.js 18+
* npm 9+
* react-native-cli 0.70+

To install Rozgar_Connect, follow these steps:
```bash
git clone https://github.com/user/Rozgar_Connect.git
```
```bash
cd Rozgar_Connect
```
```bash
npm install
```
```bash
cp .env.example .env.local
```
```bash
npm run start
```

## ⚙️ Configuration
To configure Rozgar_Connect, you will need to create a `.env` file with the following variables:
| Variable | Description |
| --- | --- |
| `REACT_NATIVE_ENV` | The environment in which the application is running |
| `NODE_ENV` | The environment in which the backend is running |
To create a `.env` file, run the following command:
```bash
cp .env.example .env.local
```
Then, update the variables in the `.env.local` file to match your environment.

## 📖 Usage
To use Rozgar_Connect, follow these steps:
1. Open the Rozgar_Connect app on your mobile device
2. Create a profile with your location and job preferences
3. Browse through the list of available jobs in your area
4. Select a job to view its details and initiate a chat with the employer
5. Discuss the job requirements and salary with the employer through the in-app chat feature
6. Accept the job offer and start working
